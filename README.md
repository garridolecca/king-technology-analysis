# King Technology - Strategic Industry Analysis

Interactive web application analyzing challenges and opportunities for **King Technology** in the pool and spa water care industry.

**Live App:** [https://garridolecca.github.io/king-technology-analysis/](https://garridolecca.github.io/king-technology-analysis/)

## Features

- **Interactive Global Map** - ArcGIS JS SDK 5.0 map showing King Technology HQ, major competitors (Fluidra, Hayward, Pentair), and high-growth APAC markets
- **Challenges Analysis** - Smart pool tech disruption, distribution model pressure, scale disadvantage, raw material costs, consumer perception
- **Opportunities Analysis** - Wellness mega-trend, Smart FROG products, Asia-Pacific expansion, commercial pools, hot tub/swim spa growth, subscription model
- **SWOT Dashboard** - Strengths, weaknesses, opportunities, and threats at a glance
- **Market Data** - $49.1B market by 2034, 6.6% CAGR, regional breakdowns
- **Secure Access** - Login-protected portal

## Tech Stack

- ArcGIS Maps SDK for JavaScript 5.0
- Calcite Design System (dark theme)
- Vanilla JavaScript (no build step)
- GitHub Pages deployment

## Market Context

The global pool and spa market is projected to reach **$49.1 billion by 2034** at 6.6% CAGR. King Technology's mineral-based FROG system (50% less chlorine) is well-positioned for the wellness trend, but faces disruption from smart pool IoT systems growing at 10.3% CAGR.
